Enlighten API Docs

Below are the available endpoints and their descriptions:

  • /: This endpoint returns the index.html template.
  • /docs: This endpoint returns the docs.html template.
  • /api: This endpoint handles GET and POST requests. It expects query parameters data and token. It returns JSON responses based on the value of the data parameter.
  • /data: This endpoint returns the data_files dictionary as a JSON response.
  • /dbr_fx: This endpoint calculates and returns the dbr_fx value as a JSON response.
  • /dbr_issuance: This endpoint separates the datasets in the "dbr" file and returns the dbr_issuance value as a JSON response.
  • /dbr_value: This endpoint calculates the dbr_value based on the dbr_fx and dbr_policy values, and returns it as a JSON response.
  • /inv_stake: This endpoint calculates and returns the inv_stake value as a JSON response.
  • /dbr_per_inv: This endpoint calculates the dbr_per_inv value based on the dbr_policy and inv_stake values, and returns it as a JSON response.
