Enlighten API Docs

Below are the available endpoints and their descriptions:

  • /dbr_policy: Retrieves the "dbr_policy" data from the endpoint "https://www.inverse.finance/api/transparency/dbr-emissions" and returns it in a JSON format after processing the data with the function dbr_policy().
  • /dbr_price: Retrieves the "dbr_price" data from the endpoint "https://www.inverse.finance/api/dbr" and returns it in a JSON format after processing the data with the function dbr_price().
  • /tvl: Retrieves the "tvl" data from the endpoint "https://www.inverse.finance/api/f2/tvl" and returns it in a JSON format after processing the data with the function tvl().
  • /firm: Retrieves the "firm" data from the endpoint "https://www.inverse.finance/api/f2/fixed-markets" and returns it in a JSON format after processing the data with the function firm().
  • /positions: Retrieves the "positions" data from the endpoint "https://www.inverse.finance/api/f2/firm-positions" and returns it in a JSON format after processing the data with the function positions().
  • /tvl_firm: Executes the function tvl_firm() which fetches the "tvl" and "firm" data from their respective endpoints, processes the data, and returns it in a JSON format.
  • /dbr_issue: Executes the function dbr_issue() which fetches multiple data sets from various endpoints, processes the data, and returns it in a JSON format.
  • /inv_stake: Executes the function inv_stake() which fetches data from various endpoints, processes the data, and returns it in a JSON format.
    • li>/dbr_inv: Executes inv_stake(), dbr_policy(), dbr_issue() to return the DBR and $ per INV per year. li>/inv_fx: Executes the function tvl_firm() and returns the price of INV.
